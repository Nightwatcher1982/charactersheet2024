# D&D 2024 角色卡创建工具

一个基于 Next.js 构建的龙与地下城 2024 版规则角色卡创建和管理工具。

## 功能特性

✨ **引导式创建流程**
- 一步步引导创建角色
- 选择职业、物种、背景
- 分配属性值、选择技能和装备

📱 **移动端友好**
- 完美适配手机和平板设备
- 响应式设计，随时随地使用

📄 **PDF 导出**
- 一键导出精美的角色卡 PDF
- 最多两页，方便打印

💾 **本地存储**
- 自动保存角色数据
- 支持创建多个角色
- 可随时编辑和删除

🎨 **现代化界面**
- 清晰直观的用户界面
- 平滑的动画过渡
- 专业的视觉设计

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **PDF 生成**: jsPDF + html2canvas
- **图标**: Lucide React

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

打开浏览器访问 [http://localhost:3000](http://localhost:3000)

### 构建生产版本

```bash
npm run build
npm start
```

## 使用说明

1. **创建新角色**
   - 点击首页的"创建新角色"按钮
   - 按照步骤填写角色信息

2. **编辑角色**
   - 在首页点击角色卡片上的编辑按钮
   - 修改后记得保存

3. **导出 PDF**
   - 在角色卡片或审核页面点击"导出 PDF"
   - 浏览器会自动下载 PDF 文件

4. **删除角色**
   - 在首页点击角色卡片上的删除按钮
   - 确认删除操作

## 创建步骤说明

1. **基本信息** - 输入角色名、等级和阵营
2. **选择职业** - 12 个核心职业可选
3. **选择物种** - 选择种族（2024版称为物种）
4. **选择背景** - 确定角色的背景故事
5. **分配属性** - 使用标准数组或手动输入
6. **选择技能** - 选择角色擅长的技能
7. **选择装备** - 配置武器、护甲和冒险装备
8. **确认完成** - 审核信息并导出

- 建议定期导出 PDF 作为备份
