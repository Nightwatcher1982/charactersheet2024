# 📚 D&D Beyond 官方规则学习和改进方案

## 📖 第一部分：官方角色创建流程

根据 D&D Beyond 2024版官方规则，角色创建有以下**标准步骤**：

### 🎯 官方5步创建流程

```
1. Choose a Class（选择职业）
   ↓
2. Determine Origin（确定起源）
   ├─ Choose Background（选择背景）
   ├─ Choose Species（选择物种）
   └─ Choose Languages（选择语言）
   ↓
3. Determine Ability Scores（确定属性值）
   ├─ Generate Scores（生成分数）
   └─ Adjust by Background（背景调整）
   ↓
4. Choose Alignment（选择阵营）
   ↓
5. Fill In Details（填写细节）
   ├─ Record Class Features
   ├─ Fill in Numbers (HP, AC, Initiative等)
   ├─ Name Character
   └─ Create Final Details
```

---

## 📋 第二部分：详细规则分析

### 1️⃣ 职业选择（Class Selection）

#### 官方流程

**步骤1：选择职业**
- 12个职业可选
- 记录职业、等级（通常从1级开始）
- 记录护甲训练（Armor Training）
- **暂不填写其他细节**（在步骤5才完整填写）

**关键点：**
- 职业是**最重要的决定**，影响后续所有选择
- 此时只选择职业本身，**不立即选择子职业**（subclass在3级+）
- 不立即选择技能、特性等

#### 官方职业概览表

| 职业 | 喜好 | 主要属性 | 复杂度 |
|------|------|---------|--------|
| Barbarian | 战斗 | 力量 | 中等 |
| Bard | 表演 | 魅力 | 高 |
| Cleric | 神祗 | 感知 | 中等 |
| Druid | 自然 | 感知 | 高 |
| Fighter | 武器 | 力量或敏捷 | 低 |
| Monk | 徒手战斗 | 敏捷和感知 | 高 |
| Paladin | 防御 | 力量和魅力 | 中等 |
| Ranger | 生存 | 敏捷和感知 | 中等 |
| Rogue | 潜行 | 敏捷 | 低 |
| Sorcerer | 力量 | 魅力 | 高 |
| Warlock | 神秘学识 | 魅力 | 高 |
| Wizard | 法术书 | 智力 | 中等 |

**注意：**
- 没有在这个阶段要求选择技能
- 没有在这个阶段要求选择职业特性
- 这些都在**步骤5（填写细节）**才处理

---

### 2️⃣ 起源确定（Origin Determination）

#### A. 背景选择（Background）

**官方结构：**
背景包含以下5个固定部分：

1. **Ability Scores（属性加值）** ⭐
   - 列出3个属性
   - 选择：一个+2、另一个+1，**或**三个各+1
   - 上限20

2. **Feat（专长）**
   - 获得一个指定的起源专长（Origin Feat）

3. **Skill Proficiencies（技能熟练）**
   - 2个**指定的**技能

4. **Tool Proficiency（工具熟练）**
   - 1个指定的工具

5. **Equipment（装备）**
   - 选择A或B
   - A：装备包
   - B：50金币

**基础规则中的4个背景示例：**

##### 侍僧（Acolyte）
- **属性**：智力、感知、魅力
- **专长**：魔法启蒙（牧师）
- **技能**：洞悉、宗教
- **工具**：书法工具
- **装备**：A或B（50GP）

##### 罪犯（Criminal）
- **属性**：敏捷、体质、智力
- **专长**：警觉
- **技能**：巧手、隐匿
- **工具**：盗贼工具
- **装备**：A或B（50GP）

##### 学者（Sage）
- **属性**：体质、智力、感知
- **专长**：魔法启蒙（法师）
- **技能**：奥秘、历史
- **工具**：书法工具
- **装备**：A或B（50GP）

##### 士兵（Soldier）
- **属性**：力量、敏捷、体质
- **专长**：野蛮攻击者
- **技能**：竞技、威吓
- **工具**：一种游戏套装
- **装备**：A或B（50GP）

**关键发现：**
- ✅ 背景的技能是**固定的**，不是可选的
- ✅ 背景提供的专长是**指定的**，不是自选的
- ✅ 装备有两个选项：A（装备包）或 B（50金币）

#### B. 物种选择（Species）

**官方结构：**
物种包含以下4个固定部分：

1. **Creature Type（生物类型）**
   - 基础规则中所有都是人形生物（Humanoid）

2. **Size（体型）**
   - 中型或小型
   - 有些物种可选（如人类、魔人）

3. **Speed（速度）**
   - 通常30尺，巨人裔35尺

4. **Special Traits（特殊特质）**
   - 物种独特能力
   - **有些特质需要选择**（如精灵血统、巨人血统等）

**关键物种特质选择：**

##### 人类（Human）
- **体型选择**：中型或小型
- **技能选择**：任意1个技能
- **专长选择**：任意1个起源专长（推荐：技能专家）

##### 精灵（Elf）
- **血统选择**：卓尔/高等精灵/木精灵
  - 不同血统提供不同戏法和法术
- **技能选择**：洞悉/察觉/生存（三选一）

##### 龙裔（Dragonborn）
- **龙类型选择**：10种龙（黑/蓝/黄铜/青铜/紫铜/金/绿/红/银/白）
  - 影响吐息武器和伤害抗性

##### 侏儒（Gnome）
- **血统选择**：森林侏儒或岩石侏儒
  - 不同血统提供不同能力

##### 巨人裔（Goliath）
- **巨人血统选择**：6种巨人（云/火/霜/丘陵/石/风暴）
  - 每种提供不同的战斗能力

##### 魔人（Tiefling）
- **体型选择**：中型或小型
- **炼狱遗产选择**：深渊/冥界/炼狱
  - 影响抗性、戏法和法术

**重要：**
- 物种的选择在**步骤2**完成
- 特质选择**不是独立的子步骤**，而是在记录物种特质时完成

#### C. 语言选择（Languages）

**规则：**
- 至少3种语言
- 通用语（Common）+ 2种标准语言
- 可以投骰或自选

### 3️⃣ 属性值确定（Ability Scores）

#### 官方流程

**步骤A：生成分数**
三种方法：

1. **标准数组**：15, 14, 13, 12, 10, 8
2. **随机生成**：投4d6取最高3个，共6次
3. **购点法**：27点，成本表如下

**购点法成本表：**
| 分数 | 成本 |
|------|------|
| 8 | 0 |
| 9 | 1 |
| 10 | 2 |
| 11 | 3 |
| 12 | 4 |
| 13 | 5 |
| 14 | 7 |
| 15 | 9 |

**步骤B：分配属性**
- 将6个分数分配到6个属性
- 考虑职业的主要属性
- 建议表格提供了每个职业的推荐分配

**步骤C：背景调整** ⭐
- 根据背景提供的3个属性
- 选择一个+2、另一个+1，或三个各+1
- 任何属性不能超过20

**关键点：**
- 属性调整在**分配后**进行
- 调整来自**背景**，不来自物种
- 这是2024版的重大变化

---

### 4️⃣ 阵营选择（Alignment）

**9种阵营：**
- 守序善良（LG）
- 中立善良（NG）
- 混乱善良（CG）
- 守序中立（LN）
- 绝对中立（N）
- 混乱中立（CN）
- 守序邪恶（LE）
- 中立邪恶（NE）
- 混乱邪恶（CE）

**官方建议：**
- 玩家角色通常不是邪恶阵营
- 选邪恶前先询问DM

---

### 5️⃣ 填写细节（Fill In Details）

**此时才处理：**

#### A. 记录职业特性
- 查看职业特性表
- 记录1级特性
- **做出特性提供的选择**（如战士的战斗风格）

#### B. 填写数字
- **豁免投骰**：能力调整值 + 熟练加值（如果熟练）
- **技能**：能力调整值 + 熟练加值（如果熟练）
- **被动察觉**：10 + 感知（察觉）检定调整值
- **生命值**：
  - 1级：职业生命骰最大值 + 体质调整值
  - 野蛮人：12 + 体质
  - 战士/圣武士/游侠：10 + 体质
  - 吟游诗人/牧师/德鲁伊/武僧/游荡者/邪术师：8 + 体质
  - 术士/法师：6 + 体质
- **先攻**：敏捷调整值
- **护甲等级（AC）**：
  - 无护甲：10 + 敏捷调整值
  - 有护甲：根据护甲规则计算
- **攻击加值**：
  - 近战：力量调整值 + 熟练加值
  - 远程：敏捷调整值 + 熟练加值
- **法术相关**：
  - 法术豁免DC：8 + 施法能力调整值 + 熟练加值
  - 法术攻击加值：施法能力调整值 + 熟练加值

#### C. 角色命名
- 自选名字

#### D. 创建最终细节
- 性别、外貌、个性等

## 🔍 第三部分：当前应用分析

### 当前流程 vs 官方流程

| 步骤 | 官方顺序 | 当前应用 | 问题 |
|------|---------|---------|------|
| 1 | 职业 | ✅ 基本信息 | ❌ 顺序不符 |
| 2 | 起源（背景+物种+语言）| ❌ 职业→物种→背景 | ❌ 顺序错误 |
| 3 | 属性值 | ✅ 属性分配 | ⚠️ 背景调整时机错误 |
| 4 | 阵营 | ⚠️ 在基本信息中 | ⚠️ 时机偏早 |
| 5 | 填写细节 | ❌ 分散在多步 | ❌ 不集中 |

### 具体问题

#### 问题1：职业选择时机和内容 ⚠️
**当前：**
- 选择职业后立即要求选择技能
- 立即要求选择职业特性（如战斗风格）

**官方：**
- 步骤1只选择职业本身
- 技能和特性选择在**步骤5**

**影响：**
- 打乱了官方的教学顺序
- 用户在还不了解属性的情况下就要选技能

#### 问题2：背景选择缺少关键信息 ❌
**当前：**
- 显示16个背景（✅ 正确）
- 显示技能、工具、装备（✅ 正确）
- ✅ 有属性加值交互界面

**缺少：**
- ❌ 背景提供的**专长**（Feat）未显示
- ❌ 装备选择（A或B）未实现
- ❌ 背景故事（Brief Narrative）未显示

**示例（侍僧）：**
```
应该显示：
✓ 属性加值：智力/感知/魅力（+3）
✓ 专长：魔法启蒙（牧师）← 缺少
✓ 技能：洞悉、宗教
✓ 工具：书法工具
✓ 装备选择：
  □ A: 书法工具、祈祷书、圣徽、羊皮纸x10、长袍、8GP
  □ B: 50GP ← 缺少
✓ 背景故事：你在神殿中服务... ← 缺少
```

#### 问题3：物种选择完整但需优化 ⚠️
**当前：**
- ✅ 10个物种齐全
- ✅ 有特性选择系统
- ✅ 特质描述详细

**可改进：**
- 特质选择的**说明性文字**不够友好
- 没有物种的**背景故事**
- 没有物种的**寿命**信息
- 没有明确说明哪些特质是**立即生效**，哪些是**等级提升后**

#### 问题4：属性分配流程不符合官方 ❌
**当前流程：**
```
选择分配方法
    ↓
分配属性（基础值）
    ↓
[在背景选择时] 背景属性加值
```

**官方流程：**
```
选择分配方法
    ↓
分配属性（基础值）
    ↓
[在属性页面] 根据背景调整 ← 应该在这里
```

**问题：**
- 背景调整和基础分配**分离**了
- 用户可能忘记已经加过背景加值
- 最终审核页面需要显示计算过程

#### 问题5：缺少语言选择 ❌
**完全缺少：**
- 没有语言选择步骤
- 没有语言显示
- Character数据结构中没有languages字段

#### 问题6：技能选择时机和方式错误 ❌
**当前：**
- 职业技能：在职业选择后立即选择
- 背景技能：自动获得（✅ 正确）

**官方：**
- 职业技能：在**步骤5**选择
- 原因：用户需要先了解属性和技能的关系

#### 问题7：缺少专长（Feat）系统 ❌
**当前：**
- 完全没有专长系统
- 背景提供的专长未显示
- 人类的"多才多艺"特质（应获得1个起源专长）未实现

## 🎯 第四部分：改进方案

### 方案A：完全遵循官方流程（推荐）⭐

#### 新流程设计

```
步骤1：欢迎和准备
├─ 角色名称（可选，可稍后填写）
└─ 说明创建流程

步骤2：选择职业 ⭐ 简化
├─ 12个职业卡片
├─ 显示：名称、描述、主要属性、复杂度
├─ 点击查看详情（特性、子职业预览）
└─ 只记录职业，不做其他选择

步骤3：确定起源
├─ 3.1 选择背景
│   ├─ 显示16个背景
│   ├─ 每个背景显示：
│   │   ├─ 故事描述
│   │   ├─ 属性选项（3个）
│   │   ├─ 提供的专长 ⭐ 新增
│   │   ├─ 技能（2个固定）
│   │   ├─ 工具（1个）
│   │   └─ 装备选择（A或B）⭐ 新增
│   └─ [子步骤] 选择装备包或金币 ⭐ 新增
│
├─ 3.2 选择物种
│   ├─ 显示10个物种
│   ├─ 每个物种显示：
│   │   ├─ 背景故事 ⭐ 新增
│   │   ├─ 体型、速度
│   │   ├─ 特质列表（标注等级要求）⭐ 改进
│   │   └─ 寿命 ⭐ 新增
│   └─ [子步骤] 物种特性选择（如果需要）
│
└─ 3.3 选择语言 ⭐ 新增
    ├─ 自动获得：通用语
    ├─ 选择2个标准语言
    ├─ 显示语言的来源说明
    └─ 职业可能提供额外语言

步骤4：确定属性值
├─ 4.1 选择生成方法
│   ├─ 标准数组
│   ├─ 购点法
│   └─ 手动输入（投骰）
│
├─ 4.2 分配基础属性
│   ├─ 使用选择的方法分配
│   ├─ 显示职业推荐
│   └─ 显示每个属性的说明
│
└─ 4.3 背景调整 ⭐ 关键改进
    ├─ 自动显示背景提供的3个属性
    ├─ 选择如何分配+3点
    ├─ 实时显示最终属性值
    └─ 公式显示：最终值 = 基础值 + 背景加值

步骤5：选择阵营
├─ 9种阵营选项
├─ 每种阵营的详细说明
└─ 个性特质建议

步骤6：填写细节 ⭐ 新增集中步骤
├─ 6.1 职业特性选择
│   ├─ 显示1级所有特性
│   ├─ 做出特性提供的选择
│   │   └─ 如战士→战斗风格
│   ├─ 选择职业技能 ⭐ 移到这里
│   └─ 记录所有特性
│
├─ 6.2 专长确认 ⭐ 新增
│   ├─ 显示背景提供的专长
│   ├─ 如果是人类，选择额外专长
│   └─ 显示专长效果
│
├─ 6.3 技能汇总 ⭐ 改进
│   ├─ 职业技能（刚选择的）
│   ├─ 背景技能（固定）
│   ├─ 物种技能（如精灵的敏锐感官）
│   └─ 专长技能（如果专长提供）
│
├─ 6.4 计算数值
│   ├─ 自动计算所有派生值
│   ├─ 生命值、AC、先攻
│   ├─ 豁免、技能加值
│   └─ 攻击加值、法术DC
│
├─ 6.5 装备确认
│   ├─ 显示背景装备（已选）
│   ├─ 职业起始装备
│   └─ 计算负重
│
├─ 6.6 法术准备（如果是施法者）
│   ├─ 选择戏法
│   ├─ 准备法术
│   └─ 记录法术位
│
└─ 6.7 最终细节
    ├─ 外貌描述
    ├─ 个性特质
    ├─ 理想、纽带、缺陷
    └─ 背景故事

步骤7：审核和完成
├─ 完整角色卡预览
├─ 所有计算公式显示
├─ 编辑任何部分
├─ 保存角色
└─ 导出PDF
```

### 方案B：渐进式改进（如果完全重构工作量太大）

#### 优先级修复

**P0 - 必须立即修复：**

1. ✅ **添加背景专长显示**
   - 在背景卡片上显示提供的专长
   - 点击专长查看详细效果
   - 自动记录到角色数据

2. ✅ **添加背景装备选择**
   - 每个背景提供A/B两个选项
   - A：装备包（列出具体物品）
   - B：50金币
   - 用户二选一

3. ✅ **添加语言选择步骤**
   - 新增独立步骤
   - 通用语（自动）+ 2个标准语言
   - 显示语言表格

4. ✅ **专长系统基础**
   - 创建专长数据结构
   - 实现起源专长
   - 背景专长自动获得
   - 人类额外专长选择

**P1 - 重要改进：**

5. ✅ **优化属性调整流程**
   - 在属性页面显示背景加值的提醒
   - 清晰显示计算过程
   - 审核页面显示：最终值（基础+背景）

6. ✅ **移动技能选择时机**
   - 职业技能选择移到"填写细节"步骤
   - 或在职业选择后添加"稍后选择技能"选项
