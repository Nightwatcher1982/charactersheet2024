# D&D 2024 角色卡创建工具 - 使用指南

## 🎮 欢迎

欢迎使用 D&D 2024 角色卡创建工具！这个工具可以帮助你轻松创建、管理和打印龙与地下城角色卡。

## 🚀 快速开始

### 1. 安装和运行

```bash
# 安装依赖
npm install --cache ./.npm-cache --legacy-peer-deps

# 启动开发服务器
npm run dev
```

然后在浏览器中访问 `http://localhost:3000`

### 2. 生产环境部署

```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

## 📱 功能说明

### 创建新角色

1. 在首页点击 **"创建新角色"** 按钮
2. 按照 8 个步骤完成角色创建：
   - **步骤 1：基本信息** - 输入角色名、等级和阵营
   - **步骤 2：选择职业** - 从 12 个核心职业中选择
   - **步骤 3：选择物种** - 选择种族（精灵、矮人、人类等）
   - **步骤 4：选择背景** - 确定角色背景（士兵、学者、贵族等）
   - **步骤 5：分配属性** - 分配力量、敏捷等六大属性
   - **步骤 6：选择技能** - 选择角色擅长的技能
   - **步骤 7：选择装备** - 配置武器、护甲和冒险装备
   - **步骤 8：确认完成** - 审核信息并完成创建

### 管理角色

- **编辑角色**：在首页点击角色卡片上的 ✏️ 编辑按钮
- **删除角色**：点击 🗑️ 删除按钮（会要求确认）
- **导出 PDF**：点击 📄 按钮导出打印版角色卡

### 导出和打印

1. 在角色列表或审核页面点击 **"导出 PDF"** 按钮
2. 浏览器会自动下载 PDF 文件
3. 打开 PDF 文件并打印（最多 2 页）

### 施法职业
- **法师**：强大的奥术施法者
- **术士**：天生的魔法使用者
- **邪术师**：与异界存在订立契约

### 混合职业
- **吟游诗人**：魔法与音乐的结合

- **德鲁伊**：自然魔法和变形能力

## 📊 属性系统

### 六大属性

1. **力量 (STR)**：物理力量、近战攻击和伤害
2. **敏捷 (DEX)**：灵活性、AC、远程攻击、先攻
3. **体质 (CON)**：生命值、耐力、抵抗疾病
4. **智力 (INT)**：学识、推理、法师施法属性
5. **感知 (WIS)**：洞察力、察觉、牧师/德鲁伊施法
6. **魅力 (CHA)**：个人魅力、领导力、部分职业施法

### 属性调整值计算

调整值 = (属性值 - 10) ÷ 2（向下取整）

例如：
- 属性值 16 = +3 调整值
- 属性值 14 = +2 调整值
- 属性值 10 = +0 调整值
- 属性值 8 = -1 调整值

### 分配方法

**标准数组**（推荐新手）
- 使用预设数值：15, 14, 13, 12, 10, 8
- 将这些数值分配到六个属性中

**手动输入**
- 自己投骰或手动输入数值
- 范围通常是 8-15（种族加值前）

## 🎯 技能系统

每个技能都关联一个属性。进行技能检定时：

**技能检定 = d20 + 属性调整值 + 熟练加值（如果熟练）**

### 常用技能

- **运动 (力量)**：攀爬、跳跃、游泳
- **杂技 (敏捷)**：平衡、翻滚、躲闪
- **隐匿 (敏捷)**：潜行、躲藏
- **奥秘 (智力)**：魔法知识
- **察觉 (感知)**：注意周围环境
- **洞悉 (感知)**：看穿谎言、理解动机
- **说服 (魅力)**：外交、谈判

## ⚔️ 战斗数据

### 护甲等级 (AC)
- 决定敌人击中你的难度
- 基础 AC = 10 + 敏捷调整值 + 护甲加值

### 生命值 (HP)
- 1 级生命值 = 职业生命骰最大值 + 体质调整值
- 例如：战士（d10）= 10 + 体质调整值

### 熟练加值
- 1-4 级：+2
- 5-8 级：+3
- 9-12 级：+4
- 13-16 级：+5
- 17-20 级：+6

## 💡 新手建议

### 第一次创建角色？

1. **职业选择**：建议选择战士或牧师（比较容易上手）
2. **属性分配**：优先提升职业主要属性
   - 战士：力量或敏捷
   - 法师：智力
   - 牧师：感知
3. **技能选择**：选择角色设定相关的技能
4. **装备选择**：根据职业熟练项选择

### 进阶技巧

- **最大化主要属性**：将最高值（15）分配给职业主要属性
- **考虑体质**：所有角色都需要体质（影响生命值）
- **平衡发展**：不要让任何属性太低（最低建议 8）
- **主题一致**：让技能、背景和装备符合角色设定

- 角色数据自动保存在浏览器本地存储中

### 注意事项
- 清除浏览器数据会丢失角色信息
- 建议定期导出 PDF 作为备份
- 更换设备需要重新创建角色

## 🐛 常见问题

### Q: 我的角色数据丢失了？
A: 检查是否清除了浏览器数据。建议定期导出 PDF 备份。

### Q: PDF 导出没有反应？
A: 确保浏览器允许下载，检查是否有弹窗拦截。

### Q: 移动端无法使用某些功能？
A: 建议使用 Chrome 或 Safari 浏览器，确保已更新到最新版本。

### Q: 如何更改已创建的角色？
A: 在首页点击角色卡片的编辑按钮，可以重新进入创建流程修改。
